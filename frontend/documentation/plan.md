# ENAULD Website Plan

The previous WorkerBull implementation has been replaced by a light-theme ENAULD website.

Current structure:

- `index.html`: ENAULD single-page site with Home, About, Services, Health, IMT, Team, Track Record, Publications and Contact sections.
- `Contact.html`: standalone ENAULD contact page.
- Legacy pages such as `Projects.html`, `Enroll.html`, `Masterclass.html`, `BookConsultation.html` and `WorkerBull.html` redirect to the new homepage or relevant section.

Future content changes should use `../../enauldthrs-website-full-content.md` and `../../colour_theme.txt` as source files.
